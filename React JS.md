# React JS

Tutorial from [here](https://reactjs.org/docs/create-a-new-react-app.html#create-react-app)

## Set up
Create the app
```sh
npx create-react-app my-app
```

Delete the contents of the src folder
```sh
cd my-app
rm -rf src
mkdir src
```

Start the application
```sh
npm start
```
